  <section>
    <h2>Was ist Schach?</h2>
    <p>
      Schach ist ein strategisches Brettspiel für zwei Spieler, das auf einem
      8×8‑Feld gespielt wird. Ziel ist es, den gegnerischen König so anzugreifen,
      dass er nicht mehr entkommen kann – das nennt man „Schachmatt“.
    </p>
    <img class="image" src="https://upload.wikimedia.org/wikipedia/commons/3/30/ChessStartingPosition.jpg" alt="Schach Startaufstellung" />
  </section>

  <section>
    <h2>Ursprung des Schachs</h2>
    <p>
      Schach entstand im 6. Jahrhundert in Indien unter dem Namen „Chaturanga“.
      Von dort verbreitete es sich nach Persien und später über die islamische Welt
      nach Europa.
    </p>
  </section>

  <section>
    <h2>Schach im Mittelalter</h2>
    <p>
      Im Mittelalter wurde Schach in Europa sehr beliebt. Viele Regeln entwickelten sich
      weiter, und das Spiel wurde zunehmend komplexer.
    </p>
  </section>
